# Repository Guidelines

This repository contains a Chainlit application using Google Gemini and MCP. When contributing:

* Use **Python 3.10+**.
* Format code with `black` and keep lines under 120 characters.
* Add docstrings to new functions and classes.
* Update or add tests under `tests/` to cover new functionality.
* Run tests with `pytest -q` and ensure they pass before committing.
* In pull request descriptions, include a brief summary of the changes and a **Testing** section with the test output. If tests cannot be run due to environment limitations, note it in the Testing section.
